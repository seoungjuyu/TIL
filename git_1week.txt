원격 ROPO 사용하기 - 정리
1. 원격 repo 와 로컬 repo 를 연결해서 내용을 반영하고 싶을 땐 어떤 방법을 써야
로컬 repo가 없고 원격 repo에 있는 내용을 가져오고 싶으면 clone을 한다. 
2. 원격 repo 와 로컬 repo 는 왜 따로 있을까? 
협업할 떄, 혹은 여러 컴퓨터를 사용한다면 하나의 원격 repo 에 여러 로컬 repo을 연결시킬수있다.
하나의 프로젝트를 동시에 작업이 가능하다
3. push 와 pull 의 개념을 원격 repo와 로컬 repo 를 포함해 그림으로 그려보세요.
